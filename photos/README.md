## How to Add Your Photos

1. **Add your photos to this folder**
   - Recommended image formats: .jpg, .jpeg, .png
   - For best results, use high-quality images

2. **Update the HTML file**
   
   Find the gallery section in `index.html` and replace the placeholder cards with your images:

   **Replace this:**
   ```html
   <div class="gallery-item">
       <div class="placeholder-card">
           <span class="placeholder-icon">üì∑</span>
           <p>Add your favorite photo here</p>
       </div>
   </div>
   ```

   **With this:**
   ```html
   <div class="gallery-item">
       <img src="photos/your-photo-name.jpg" alt="Our Memory">
   </div>
   ```

3. **Special Surprise Photo**
   
   Add a special photo named `surprise.jpg` in this folder. This will be revealed when the gift box is fully unwrapped!

## Example Photo Names
- `surprise.jpg` (Required for gift box)
- `memory1.jpg`
- `memory2.jpg`
- `together.jpg`
- `special-moment.jpg`

## Tips
- Use descriptive alt text for each image
- Keep file sizes reasonable (under 2MB per image) for faster loading
- You can add as many photos as you want to the gallery!
